Bitmap2fnt converter v1 - 110313

This free tool will create a .fnt file to use with a bitmap font file.


1- Click "Load bitmap" and select the bitmap font file (png, jpeg, bmp, ..),

2- Type width and height of one character in your *fixed* size bitmap,

3- Type first and last characters displayed in your bitmap font,

4- Type fontname,

5- Click Convert.

A new file <fontname>.fnt is created in the bitmap font file directory.



To run from command prompt, type :

java -jar "bitmap2fnt-v1.jar"
